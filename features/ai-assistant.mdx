# General

Our AI Assistant is a living machine that gets smarter and faster each day.
It will help you design and create RevoEngine solutions with ease, even when you have little experience.

# Threads

Categorize your conversations into simple **Threads**, bound to a specific topic to get the best answers.
Each thread can have an unlimited number of messages where you can chat using human language, we support 90+ languages.
